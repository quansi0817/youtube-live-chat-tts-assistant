Why manual setup?
Due to browser security (CORS), we cannot automatically inject scripts into YouTube pages. You need to manually run a script in the YouTube chat window console.

Troubleshooting

    
    

        Script not working?

        - Make sure you're on a YouTube live chat page (youtube.com/live_chat)

        - Make sure the chat is actually live and active

        - Check the console for error messages

        - Try refreshing the page and running the script again
    

    
    

        Messages not appearing in main app?

        - Make sure the main application window is still open

        - Check that popup blockers aren't blocking the connection

        - Try disconnecting and reconnecting
